# Logo usage

## Which logo for which background

| Background | Use this file (PNG) | Use this file (SVG) |
|---|---|---|
| Forest Green `#223728` | `Logos/PNG/ShuffleeGo_logo para fundo verde.png` (horizontal) or `...vertical.png` | `Logos/SVG/ShuffleeGo_logo para fundo verde vertical.svg` |
| Dark photo (any moody photographic background) | `Logos/PNG/ShuffleeGo_negativo.png` (white reverse) | `Logos/SVG/ShuffleeGo_negativo.svg` |
| Light / cream / white | `Logos/PNG/ShuffleeGo_logo.png` or `ShuffleeGo_positivo.png` | `Logos/SVG/ShuffleeGo_logo.svg` |
| Instagram profile picture or favicon only | _(use the brand's official Instagram icon — request from Alê if needed)_ | _(same)_ |

**Rule:** when in doubt, SVG. PNG is only for fixed-size raster contexts (slides, small social tiles). Anything that might be scaled or printed → SVG.

## Clearspace

Reserve a margin equal to the **height of the "S" in the wordmark** around the entire logo. No other graphic, text, or edge of frame may enter this clearspace.

```
   |←─ S height ─→|
─────────────────────────────────────
│                                     │
│      ┌─────────────────────┐        │
│      │                     │        │
│      │   Shuffle & Go      │        │
│      │                     │        │
│      └─────────────────────┘        │
│                                     │
─────────────────────────────────────
```

## Minimum sizes

- **Horizontal logo:** 120 px wide on screen / 25 mm wide in print. Below this, switch to the vertical lockup or the Instagram icon.
- **Vertical logo:** 60 px wide on screen / 15 mm wide in print.
- **Cabinet Grotesk type lockup:** never below **14 pt**.

## What you may NOT do

1. **Recolor** the logo (any color other than the canonical files supplied).
2. **Outline** or stroke the logo.
3. **Drop-shadow** the logo.
4. **Gradient-fill** or overlay-blend the logo.
5. **Stretch, skew, or rotate** the logo.
6. **Separate the mark from the wordmark** — the lockup ships intact.
7. **Replace the tagline** with the logo or use the tagline (*"Cards in Hand, World Ahead"*) in isolation as a brand mark.
8. **Re-create** the logo from scratch — always use the supplied files.

## Co-branding rules

When pairing the Shuffle and Go logo with a partner logo:

- **Position:** side-by-side horizontally, separated by a thin vertical line.
- **Separator:** Aged Gold `#be9a50`, 1–2 pt thick, height matched to ~80% of the taller logo's height. Centered vertically between the logos.
- **Hierarchy:** equal weight by default. If hosting the partnership, Shuffle and Go logo on the left; if guesting, on the right.
- **Partner color treatment:** the partner's logo keeps its own color treatment — do not force it into the Shuffle and Go palette.
- **Clearspace:** each logo retains its own clearspace inside the lockup. The separator does not count as filling the clearspace.
- **Naming in copy:** "Shuffle and Go × [Partner Name]" — use a multiplication symbol `×` (Unicode U+00D7), not the letter `x`.
