# Shuffle and Go — Partner Brand Kit

Welcome — thanks for partnering with **Shuffle and Go**. This kit contains everything you need to represent the brand correctly in shared assets: thumbnails, event graphics, social posts, sponsorship reels, co-branded designs.

If anything below is unclear, reach out to Alexandre (Alê) — alemeneghessosilva@gmail.com — **before** publishing.

---

## What's in this kit

- **`README.md`** — this file. Start here.
- **`BRAND-SUMMARY.md`** — one-page brand context (identity, premise, voice, what we are and aren't).
- **`LOGO-USAGE.md`** — clearspace, minimum sizes, co-branding rules, do's and don'ts.
- **`Logos/PNG/`** — raster logos for fixed-size contexts (web, social, slides).
- **`Logos/SVG/`** — vector logos for any context where scaling matters (print, signage, large displays).
- **`colors-and-fonts.md`** — hex codes, CMYK, Pantone, and the type stack.

## Quick start

**For a single co-branded image:**
1. Open `BRAND-SUMMARY.md` — read the "Identity" + "Hard nos" sections (2 minutes).
2. Pick the right logo from `Logos/`:
   - Co-branding **on Forest Green background** → `ShuffleeGo_logo para fundo verde.png` (or `.svg`)
   - Co-branding **on light background** → `ShuffleeGo_logo.png` / `ShuffleeGo_positivo.svg`
   - Co-branding **on a dark photo** → `ShuffleeGo_negativo.png` (white reverse)
3. Apply `LOGO-USAGE.md` clearspace + minimum size rules.
4. Use **Aged Gold `#be9a50`** as the only accent — never as wallpaper.

**For a sponsorship video or longer campaign:**
- Send Alê the script / concept before production. We'll align on voice (stakes over mechanics), language defaults (English primary, Portuguese situational), and pillar fit (competition / journey / person).

## What you can do without asking

- Use any logo file in this kit, **unmodified**, in proper clearspace, at appropriate size.
- Use the four brand colors and brand fonts in co-branded assets.
- Reference the channel by its handle `@shuffleandgo` or its full name `Shuffle and Go`.

## What requires a quick check first

- **Co-branding compositions** — send a preview before publish; we'll confirm hierarchy and accent placement.
- **Audio / VO** — any narration that pretends to be Alê's voice is a hard no. AI-generated voiceover is a hard no even when it doesn't pretend.
- **Claims** — "champion," "World," "Galactic" map to specific real tournament results. Don't apply them generically.
- **Language flips** — if you'd like Portuguese copy, let us know; we'll review the translation.

## What is not allowed

- Logo manipulation: recoloring, outlining, drop-shadowing, gradients, stretching, skewing, rotating, or separating the mark from the wordmark.
- The tagline (*"Cards in Hand, World Ahead"*) used as a logo on its own.
- Pure black `#000000` as a background substitute for Forest Green.
- Neon, saturated "gaming" color palettes as the primary look.
- "Gamer" fonts, novelty display fonts, Star Wars title fonts, hand-letter approximations.
- Game-mechanics jargon in user-facing copy where stakes language would land harder.
